Java和智能合约的交互

web3j

web3j is a lightweight, highly modular, reactive, type safe Java and Android library for working with Smart Contracts and integrating with clients (nodes) on the Ethereum network.
web3j 是一个轻量级、模块化、响应式的Java 和 android 库，可以用来和智能合约以及以太坊节点进行交互。

 
下边根据官方教程讲解搭建一个java工程 (代码详见官方样例)，并与智能合约进行交互：

Getting started

 导入web3j 包

<dependency>
  <groupId>org.web3j</groupId>
  <artifactId>core</artifactId>
  <version>3.2.0</version>
</dependency>

启动以太坊节点

 启动私链节点可以 参考前文《以太坊私链搭建教程》 。如果使用infura服务可跳过此步骤。

与以太坊节点交互

  • 构建web3j实例

 使用私链节点

Web3j web3j = Web3j.build(new HttpService("http://192.168.xxx.xxx:8545"));

 当然也可以使用Infura提供的云节点服务，那样子就不用启动以太坊私链节点了。只需要在代码里这样使用：

Web3j web3 = Web3j.build(new InfuraHttpService("https://ropsten.infura.io/your-token"));

 Infura需要申请访问token，移步官网申请：https://infura.io/

  • 构建凭据
// FIXME: Generate a new wallet file using the web3j command line tools https://docs.web3j.io/command_line.html

Credentials credentials = WalletUtils.loadCredentials("<password>","/path/to/<walletfile>");

  password是你的以太坊帐号密码，如果是私链节点，walletfile可以在数据目录（data/keystore/）下找到。

  另外也可以直接用私钥构建凭据:

Credentials credentials = Credentials.create("92a9470c8597ce44c5fd901363541c0c41e341d2c1fe719e6bf20f2557axxxxx");
  • 发起交易
TransactionReceipt transferReceipt = Transfer.sendFunds(web3j, credentials,"0xe80fbabee07b259fe36a950ef384abf401806eef",BigDecimal.ONE, Convert.Unit.WEI).send();
  • 合约部署
Greeter contract = Greeter.deploy(
                web3j, credentials,
                ManagedTransaction.GAS_PRICE, Contract.GAS_LIMIT,
                "Hello blockchain world!").send();
  • 调用合约
TransactionReceipt transactionReceipt = contract.newGreeting("Well hello again").send();
  • 监听Event
for (Greeter.ModifiedEventResponse event : contract.getModifiedEvents(transactionReceipt)) {
            log.info("Modify event fired, previous value: " + event.oldGreeting
                    + ", new value: " + event.newGreeting);
            log.info("Indexed event previous value: " + Numeric.toHexString(event.oldGreetingIdx)
                    + ", new value: " + Numeric.toHexString(event.newGreetingIdx));
        }

编译合约包装代码

 从官方样例我们可以看到，java代码和智能合约的交互是通过rpc调用来实现的。那在实际开发中，智能合约采用solidity语言开发，如何从智能合约代码生成相对应的rpc包装代码呢？
 这里用到两个工具web3j和solc 。官方仓库都提供编译好的工具，直接下载即可使用，自行源码编译容易踩坑，不建议。
 编译包装代码分为两步：

  • 用solc编译合约代码获得 bin 和 abi
$ solc <contract>.sol --bin --abi --optimize -o <output-dir>/
  • 用web3j编译bin和abi 获得包装代码
web3j solidity generate /path/to/<smart-contract>.bin /path/to/<smart-contract>.abi -o /path/to/src/main/java -p com.your.organisation.name

 另外如果你使用的truffle框架，那么可以利用项目build目录下的json文件，简化生成包装代码的步骤，如下：

cd /path/to/your/web3j/java/project
$ web3j truffle generate /path/to/<truffle-smart-contract-output>.json -o /path/to/src/main/java -p com.your.organisation.name
